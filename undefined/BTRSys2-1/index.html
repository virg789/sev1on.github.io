BTRSys2.1

Posted on 2021-07-24 | In Linux Walkthrough
This machine is exploited by guessing a Wordpress user's password and gaining a reverse shell using Wordpress theme template files. It is escalated by leaking database user's credentials, and then leaking and cracking the root user's password on the system.

Exploitation

Guessing User’s Password

Before we start brute-forcing the discovered users’ credentials in the application, we can employ some easy guesswork to find that the password for the admin user is also admin. We are able to successfully log in at http://192.168.120.40/wordpress/wp-login.php as admin:admin.
