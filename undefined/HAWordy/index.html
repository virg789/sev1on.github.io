HAWordy

Posted on 2021-07-16 | In Linux Walkthrough
This machine is exploited via a file upload vulnerability in a Wordpress plugin, leading to remote code execution. It is escalated via misconfigured SUID permissions on the **wget** binary, which allows the attacker to overwrite the **/etc/passwd** file and introduce a new root user.
